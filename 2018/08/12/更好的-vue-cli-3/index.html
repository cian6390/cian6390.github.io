什麼是 vue-cli ?

vue-cli 用於建構一個完整以vuejs為專案基底的專案

  • 通過CLI(command-line interface)的交互快速搭建專案架構
  • 搭配@vue/cli與@vue/cli-service-global實現零配置專案搭建
  • @vue/cli-service開發運行環境包含以下幾個特性
    • 可更新
    • 建構於webpack，擁有合理的預設值
    • 通過項目中的配置加以設定
    • 可使用插件(plugins)擴充

CLI

安裝全域套件@vue/cli之後系統就會有vue指令可以使用，在v3後我們除了使用vue命令建構專之外也多了vue ui命令來建立、管理專案。

CLI Service

@vue/cli-service 是個全域開發依賴套件，它會自動被安裝於使用@cue-cli所建構的專案之中。

而CLI Service建構於webpack與webpack-dev-server，包含：

  • 核心的service、loaders與其他CLI Plugins
  • 一個適用於大多數app的webpack設定
  • 命令serve, build, inspect 命令來自vue-cli-service

CLI Plugins

CLI Plugins是選擇性的功能，例如 Bbabel/TypeScript, ESLint, Test etc… 透過前綴@vue/cli-plugin-(for build-in plugins) 或是 vue-cli-plugin(for community plugins)來安裝plugin。
當使用vue-cli-service的命令，會自動地執行package.json裡頭的loads與 CLI Plugins。

安裝

nodejs版本至少要8.9以上，但建議8.11.0以上

若環境已經存在(1.x或 2.x)，請先執行npm uninstall vue-cli -g 移除。
